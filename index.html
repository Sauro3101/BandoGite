Sito web gite Checchetto

Questo sito web permette di inserire dei viaggi, delle agenzie e poi di registrare varie offerte valutando ciò che viene offerto con un punteggio ed infine di vedere la classifica delle offerte che hanno il numero maggiore di punti.

Per testare il sito è necessario importare il database. Il file per l'importazione si trova nella directory del sito ed è denominato db_bandogite.sql. Questo database contiene già dei valori di esempio utilizzati per delle prove precedenti.

Ci sono due tipi di utenti, un utente "amministratore" e un utente "semplice". Sono stati creati vari utenti, per i test utilizzare i seguenti account:

Utente amministratore: admin admin

Utente semplice: MarioRossi Password123

L'utente amministrazione ha la possibilità di inserire ed eliminare offerte, agenzie e viaggi, mentre l'utente semplice ha la sola possibilità di inserire ed eliminare le offerte sulla base delle agenzie edei viaggi inseriti dall'amministratore. Entrambi gli utenti hanno la possibilità di visualizzare la classifica.
